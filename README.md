# Sko-Nexus v3

Sistema Operativo de Orquestación Inteligente basado en el protocolo SDD (Specify, Design, Develop).

## 🚀 Estructura del Proyecto

- `🧠 mcp/`: Servidor MCP modular para herramientas y agentes.
- `🚀 bin/`: CLI principal (`sko.js`).
- `💎 prisma/`: Gestión de base de datos y esquemas Zod.
- `🌐 webapp/`: Interfaz de usuario construida en Next.js.
- `📂 assets/`: Conocimiento, habilidades (Skills) y protocolos.
- `🐳 docker/`: Configuración de infraestructura (MariaDB, Dockerfiles).

## 🛠️ Comandos Rápidos

- `npm run dev:webapp`: Inicia el entorno de desarrollo de la webapp.
- `npm run db:studio`: Abre la interfaz de Prisma para gestionar la base de datos.
